# 作用

- 自动安装GUI软件

- 自动配置terminal工具

- 自动创建常用别名

- 自动安装了.oh-my-zsh，自动补全、代码高亮、git命令增加,z快速进入目录

- 设置了terminal自动补全时不区分大小写
