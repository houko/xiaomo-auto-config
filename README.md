# macos

安装xcode

## 注意事项

- 在默认终端下执行命令
- 安装时需要多次输入管理员用户密码

## install

bash -c "$(curl -fsSL https://raw.githubusercontent.com/houko/xiaomo-auto-config/master/macos/install.sh)"

## uninstall

bash -c "$(https://raw.githubusercontent.com/houko/xiaomo-auto-config/master/macos/uninstall.sh)"
